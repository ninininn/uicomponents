# Usage API

初始化 Slider 元件，帶入參數 `parentDOM` 及 `options` 物件，並設定內部屬性。
(兩參數皆為可選值，若沒有帶入則會套用內部預設值)

```javascript
new Slider(parentDOM, (options = {}));
```
    >### parameters 參數設定
    <span className="sb-unstyled font-black">parentDOM</span>
    <span className="sb-unstyled data-type element">HTMLElement</span>
    ```html
    <!--  要放入的container -->
    <div id="slider_1"></div>
    ```
    ```javascript
    const sliderParent = document.getElementById("slider_1");
    const sliderInstance = new Slider(sliderParent, {initValue: 79});
    ```
    <span className="sb-unstyled font-black"> options</span>
    <span className="sb-unstyled data-type">object</span>


    各屬性傳入型別可參考上方 Control 說明，以下 option 設定值為內部預設值。
    ```javascript
    const sliderOptions = {
      min:1,
      max:100,
      initValue:0,
      step:1,
      range:false,
      theme:'var(--color-primary-500)',
      thumbImg:null,
      classes:['slider'],
      handler:null,
      disabled:false
    }
    ```

    >### methods 方法

     <p className="sb-unstyled font-bold text-yellow-500">Quick reference</p>
     <div className="sb-unstyled table-container overflow-hidden">
      <table className="sb-unstyled table-content">
        <thead>
          <tr className="text-center">
            <td>methods</td><td>descriptions</td><td>inherit</td>
          </tr>
        </thead>
        <tbody>
          <tr><td className="font-bold">`.getElem()`</td><td>回傳該元件渲染的 DOM 節點元素</td><td>`BaseComponent`</td></tr>
          <tr><td className="font-bold">`.getValue()`</td><td>回傳該元件的 slider 數值</td></tr>
          <tr><td className="font-bold">`.setValue(val)`</td><td>傳入要設定的 slider 數值，如果是 range-type 則傳入 `[ startVal, endVal ]`</td></tr>
          <tr><td className="font-bold">`.changeTheme(HEXstring)`</td><td>更改主題色</td></tr>
          <tr><td className="font-bold">`.setDisabled(boolean)`</td><td>更改顯示/操作與否</td></tr>

        </tbody>
      </table>
     </div>

      <span className="sb-unstyled methods">.getElem()</span>
      ``` javascript
        let sliderElem = Slider.getElem(); // return an DOM HTMLElement
      ```
      <span className="sb-unstyled methods">.getValue()</span>
      ``` javascript
        let value = Slider.getValue(); // 30
      ```
      <span className="sb-unstyled methods">.setValue(val)</span>
      ``` javascript
        Slider.setValue(45);
      console.log(Slider.getValue()); // 45
      ```
      <span className="sb-unstyled methods">.changeTheme(HEX)</span>
      ``` javascript
        Slider.changeTheme("#878787");
      ```
      <span className="sb-unstyled methods">.setDisabled(boolean)</span>
      ``` javascript
        Slider.setDisabled(true); //Slider will become disabled
      ```

    >### properties 內部屬性 <span class="sb-unstyled data-type readonly">Read-only</span>
    <p className="sb-unstyled alert-msg">可以操作的屬性皆須使用methods來更改</p>
