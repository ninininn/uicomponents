<span className="sb-unstyled methods">.addClass(element, classes)</span>
 - **Parameters**  
    - **element<span className="text-red-600">*</span>** DOM元素
    <span className="sb-unstyled data-type element">HTMLElement</span>
    - **classes<span className="text-red-600">*</span>** 要加入的 classes
    <span className="sb-unstyled data-type arr">array</span>

 在指定DOM節點上加入指定classes

``` javascript
 UIUtils.addClass(this._elem,["container","absolute"]); 
```  

<span className="sb-unstyled methods">.removeClass(element, classes)</span>
 - **Parameters**  
    - **element<span className="text-red-600">*</span>** DOM元素
    <span className="sb-unstyled data-type element">HTMLElement</span>
    - **classes<span className="text-red-600">*</span>** 要移除的 classes
    <span className="sb-unstyled data-type arr">array</span>

 在指定DOM節點上移除指定classes

``` javascript
 UIUtils.removeClass(this._elem,["container","absolute"]); 
```

<span className="sb-unstyled methods">.clearClass(element, excludeClasses)</span>
 - **Parameters**  
    - **element<span className="text-red-600">*</span>** DOM元素
    <span className="sb-unstyled data-type element">HTMLElement</span>
    - **excludeClasses** 除外classes
    <span className="sb-unstyled data-type arr">array</span>

 移除指定節點上的所有classes(除了指定excludeClasses)

``` javascript
 UIUtils.clearClass(this._elem,["name_id"]); 
```

<span className="sb-unstyled methods">.setText(element, text)</span>
 - **Parameters**  
    - **element<span className="text-red-600">*</span>** DOM元素
    <span className="sb-unstyled data-type element">HTMLElement</span>
    - **text<span className="text-red-600">*</span>** 文字內容
    <span className="sb-unstyled data-type str">string</span>

 設定指定DOM節點的textContent內容

``` javascript
 UIUtils.setText(this._elem,"測試文字"); 
```

<span className="sb-unstyled methods">.setAttribute(element, attributeName, attributeValue)</span>
 - **Parameters**  
    - **element<span className="text-red-600">*</span>** DOM元素
    <span className="sb-unstyled data-type element">HTMLElement</span>
    - **attributeName<span className="text-red-600">*</span>** 屬性名稱
    <span className="sb-unstyled data-type str">string</span>
    - **attributeValue<span className="text-red-600">*</span>** 要設定的值
    <span className="sb-unstyled data-type str">string</span>

 設定指定DOM節點的自訂`data-`屬性

``` javascript
 UIUtils.setAttribute(this._elem,"initialize","true"); 
```

<span className="sb-unstyled methods">.setProperty(element, propertyName, propertyValue)</span>
 - **Parameters**  
    - **element<span className="text-red-600">*</span>** DOM元素
    <span className="sb-unstyled data-type element">HTMLElement</span>
    - **propertyName<span className="text-red-600">*</span>** CSS propertyName
    <span className="sb-unstyled data-type str">string</span>
    - **propertyValue<span className="text-red-600">*</span>** 要設定的值
    <span className="sb-unstyled data-type str">string</span>

 設定指定DOM節點的自訂css-property屬性

``` javascript
 UIUtils.setProperty(this._elem,"--theme","#878787"); 
```

<span className="sb-unstyled methods">function hexTorgb(hexString)</span>
 - **Parameters**  
    - **hexString<span className="text-red-600">*</span>** HEX色碼
    <span className="sb-unstyled data-type str">string</span>
 - **Returns**

``` javascript
  let newColor = hexTorgb("#878787"); //rgb(135,135,135)
```
