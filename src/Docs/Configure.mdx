<span className="sb-unstyled methods">.addClass(element, classes)</span>
在指定DOM節點上加入指定classes

- **Parameters**
  - **element<span className="text-red-600">\*</span>** DOM元素
    <span className="sb-unstyled data-type element">HTMLElement</span>
  - \*\*classes
    <span className="text-red-600">*</span>** 要加入的 classes
    <span className="sb-unstyled data-type arr">array</span>

```javascript
Dom.addClass(this._elem, ["container", "absolute"]);
```

<span className="sb-unstyled methods">.removeClass(element, classes)</span>
在指定DOM節點上移除指定classes - **Parameters** - **element
<span className="text-red-600">*</span>** DOM元素
<span className="sb-unstyled data-type element">HTMLElement</span>- **classes
<span className="text-red-600">*</span>** 要移除的 classes
<span className="sb-unstyled data-type arr">array</span>

```javascript
Dom.removeClass(this._elem, ["container", "absolute"]);
```

<span className="sb-unstyled methods">
  .clearClass(element, excludeClasses)
</span>
移除指定節點上的所有classes(除了指定excludeClasses) - **Parameters** - **element
<span className="text-red-600">*</span>** DOM元素
<span className="sb-unstyled data-type element">HTMLElement</span>-
**excludeClasses** 除外classes
<span className="sb-unstyled data-type arr">array</span>

```javascript
Dom.clearClass(this._elem, ["name_id"]);
```

<span className="sb-unstyled methods">.setText(element, text)</span>
設定指定DOM節點的textContent內容 - **Parameters** - **element
<span className="text-red-600">*</span>** DOM元素
<span className="sb-unstyled data-type element">HTMLElement</span>- **text
<span className="text-red-600">*</span>** 文字內容
<span className="sb-unstyled data-type str">string</span>

```javascript
Dom.setText(this._elem, "測試文字");
```

<span className="sb-unstyled methods">
  .setAttribute(element, attributeName, attributeValue)
</span>
設定指定DOM節點的自訂`data-`屬性 - **Parameters** - **element
<span className="text-red-600">*</span>** DOM元素
<span className="sb-unstyled data-type element">HTMLElement</span>-
**attributeName<span className="text-red-600">*</span>** 屬性名稱
<span className="sb-unstyled data-type str">string</span>- **attributeValue
<span className="text-red-600">*</span>** 要設定的值
<span className="sb-unstyled data-type str">string</span>

```javascript
Dom.setDataAttr(this._elem, "initialize", "true");
```

<span className="sb-unstyled methods">
  .setProperty(element, propertyName, propertyValue)
</span>
設定指定DOM節點的自訂css-property屬性 - **Parameters** - **element
<span className="text-red-600">*</span>** DOM元素
<span className="sb-unstyled data-type element">HTMLElement</span>-
**propertyName<span className="text-red-600">*</span>** CSS propertyName
<span className="sb-unstyled data-type str">string</span>- **propertyValue
<span className="text-red-600">*</span>** 要設定的值
<span className="sb-unstyled data-type str">string</span>

```javascript
Dom.setProp(this._elem, "--theme", "#878787");
```

### 其他類別

<span className="sb-unstyled methods">
  function clamp(value, min = 0, max = 1)
</span>
檢視數值是否在目標區間 - **Parameters** - **value
<span className="text-red-600">*</span>** 目標值
<span className="sb-unstyled data-type arr">number</span>- **min** 區間最小值
<span className="sb-unstyled data-type arr">number</span>- **max** 區間最大值
<span className="sb-unstyled data-type arr">number</span>- **Returns**
如果目標值在 min, max 區間則回傳該目標值；小於 min 則回傳 min、大於 max 則回傳
max

```javascript
let clampNum = clamp(0.56); //0.56
let clampNum2 = clamp(1.5, 0, 1); //1
```
