<span className="sb-unstyled methods">.addClass(el, classes)</span> **在指定DOM節點上加入指定classes**

- **Parameters**
  - **el<span className="text-red-600">\*</span>** DOM元素
    <span className="sb-unstyled data-type element">HTMLElement</span>
  - **classes
    <span className="text-red-600">*</span>** 要加入的 classes
    <span className="sb-unstyled data-type arr">array</span>

```javascript
Dom.addClass(this._elem, ["container", "absolute"]);
```

<span className="sb-unstyled methods">.removeClass(el, classes)</span> **在指定DOM節點上移除指定classes**

  - **classes
<span className="text-red-600">*</span>** 要移除的 classes
<span className="sb-unstyled data-type arr">array</span>

```javascript
Dom.removeClass(this._elem, ["container", "absolute"]);
```

<span className="sb-unstyled methods">.clearClass(el, excludeClasses)</span> **移除指定節點上的所有classes(除了指定excludeClasses)** 
 - **Parameters** 
    - **el<span className="text-red-600">*</span>** DOM元素

    - **excludeClasses** 除外classes
  <span className="sb-unstyled data-type arr">array</span>

```javascript

```

# 其他類別
### clamp
<span className="sb-unstyled methods">clamp(value, min = 0, max = 1)</span> **檢視數值是否在目標區間**  
- **Parameters** 
  - **value
  <span className="text-red-600">*</span>** 目標值
  <span className="sb-unstyled data-type num">number</span>
  - **min** 區間最小值
  <span className="sb-unstyled data-type num">number</span>
  - **max** 區間最大值
  <span className="sb-unstyled data-type num">number</span>
- **Returns**  
如果目標值在 min, max 區間則回傳該目標值；小於 min 則回傳 min、大於 max 則回傳 max

```javascript
let clampNum = clamp(0.56); //0.56
let clampNum2 = clamp(1.5, 0, 1); //1
```

- **Returns**  <span className="sb-unstyled data-type obj">object</span>  
`{ device, size }`

```javascript
// other opereations...
checkDevice();
```

# 色彩相關
### Color class
<span className="sb-unstyled methods">Color</span> 以HSLA屬性分別儲存數值之物件類別 `Color.js`    
- **Properties** 
  - **base** <span className="sb-unstyled data-type object">object</span>   
    - **h
    <span className="text-red-600">*</span>** Hue
    <span className="sb-unstyled data-type num">number</span>
    - **s** Saturation
    <span className="sb-unstyled data-type num">number</span>
    - **l** Lightness
    <span className="sb-unstyled data-type num">number</span>
    - **a** Alpha
    <span className="sb-unstyled data-type num">number</span>
  - **luminance** 視覺亮度
  <span className="sb-unstyled data-type num">number</span>  
- **Methods** 
  - **.toRgb()** 回傳對應RGB模式色碼
  - **.toHex()** 回傳對應HEX模式色碼
  - **.toHsl()** 回傳對應HSL模式色碼  

### Palette
<span className="sb-unstyled methods">Palette(config)</span> 隨機產生一組/單一色碼，色碼模式可以是 hsl | rgb | hex `Color.js`
